## Task2
W pliku app.py znajduje się aplikacja Flask oraz wymagane endpointy, które zwaracają odpowiedź w formacie JSON.
Przykładowa odpowiedź zawiera pole message, oraz success.
Przykładowa odpowiedź:
{
  "data": [],
  "message": "Users fetched successfully.",
  "success": true
}
W pliku maria_operations znajdują się metody z operacjami na zahardcodowanej bazie danych, których używają controllery z pliku app.py.
